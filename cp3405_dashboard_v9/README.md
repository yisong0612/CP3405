CP3405 Smart Stock Dashboard v5

How to run:
1. Install Python 3.11 or 3.12.
2. Open this folder.
3. Double-click start_web.bat.
4. Keep the terminal open.

This version includes:
- Minimal overview page
- Finance-style detail page with sidebar widgets
- News and watchlist page
- Chinese / English switch
- Light / dark mode and accent color
- Real historical prices from Yahoo Finance
